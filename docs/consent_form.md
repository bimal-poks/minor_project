# Consent for Facial Data Collection — Smart Attendance System (Minor Project)

We (Amit Acharya, Anish Khadka, Bimal Pokharel, BE Electronics & Computer 
Engineering, Pulchowk Campus) are building a Smart Attendance System as our 
BE minor project, using facial recognition for automated attendance marking.

**What we're collecting:** 2-3 photos of your face, taken during class sessions.

**How it will be used:**
- Photos will be processed to generate a mathematical "face embedding" 
  (a set of numbers representing facial features, not a visual image).
- These embeddings — not the raw photos — will be stored in our project 
  database and used to match your face during attendance testing.
- Raw photos will be kept only temporarily during development and deleted 
  once embeddings are generated, or stored securely and not shared publicly.

**What it will NOT be used for:**
- No data will be shared outside this academic project.
- No data will be published, sold, or used for any purpose beyond this 
  minor project and its evaluation.

**Your rights:**
- Participation is voluntary.
- You may ask to have your data removed from the project at any time.
